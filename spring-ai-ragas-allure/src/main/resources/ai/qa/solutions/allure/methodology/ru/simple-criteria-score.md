# Метрика Simple Criteria Score (Оценка по критерию)

## Глоссарий

- **Критерий** — пользовательское правило для оценки качества ответа
- **Агентная система** — система на основе LLM, генерирующая ответы

## Описание

Метрика оценивает ответ агентной системы по заданному критерию на непрерывной шкале от 0.0 до 1.0. В отличие от бинарной метрики Aspect Critic, позволяет получить градуированную оценку.

## Пример

**Критерий:** «Техническая точность ответа»

**Ответ агентной системы:** «Эйфелева башня была построена в 1889 году. Её высота примерно 330 метров. Её спроектировал Гюстав Эйфель.»

**Анализ:**
- Год верный (1889) ✓
- Высота верная (330м) ✓
- Проектировщик верный ✓

**Скор:** **0.95 (95%)**

---

**Ответ агентной системы:** «Эйфелева башня была построена около 1900 года. Это самое высокое сооружение в Париже.»

**Анализ:**
- Год неверный (1889, не 1900) ✗
- Утверждение о высоте спорно ⚠

**Скор:** **0.40 (40%)**

## Интерпретация скора

|  Скор   |                Значение                |
|---------|----------------------------------------|
| 90-100% | Полное соответствие критерию           |
| 70-90%  | Преимущественное соответствие критерию |
| 50-70%  | Частичное соответствие критерию        |
| 0-50%   | Несоответствие критерию                |

## Алгоритм

1. **Определение критерия** — задаётся правило оценки
2. **Анализ** — LLM оценивает ответ по критерию
3. **Присвоение скора** — возвращается оценка от 0.0 до 1.0 с обоснованием

## Примеры критериев

- **Техническая точность** — фактическая корректность информации
- **Полезность** — практическая ценность для пользователя
- **Ясность** — понятность и структурированность изложения
- **Полнота** — степень раскрытия темы

## Формула

```
score = LLM_assigned_score (от 0.0 до 1.0)
```

LLM присваивает непрерывную оценку на основе анализа соответствия критерию.
